## デバッグ
データベースを立ち上げてから、バックエンドを起動

データベースを起動
`docker compose up -d`

データベースを初期化
`docker compose down -v`

spring boot を起動
`./mvnw spring-boot:run`

サーバーをデバッグ用に起動（ブレイクポイント付き）
F5 Debug Application

## インストール
spring boot 4.0.0, maven, Java 24
https://start.spring.io

VSCode 拡張機能
Extension Pack for Java 36,018,522
